# Dissertation

## Code setup

- python `3.11.0` is required, newer versions do not work

#### Clone repo

```bash
git clone  --recurse-submodules git@github.com:hirok2005/disseration.git
```

#### Setup virtual environment

```bash
python -m venv venv
source venv/bin/activate
```

#### Install DSNS

```bash
cd DSNS
pip3 install --editable .
```
